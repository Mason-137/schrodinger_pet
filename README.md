# schrodinger_pet
《Schrodinger Pet》是一款集桌面宠物、系统监控与效率工具于一体的量子猫助手，它不仅能够以可爱的动态形象常驻桌面，还支持多主题皮肤、粒子动画、鼠标视线追踪与丰富表情互动，为桌面带来沉浸式陪伴体验。同时，软件内置 CPU / 内存 / 温度监控、网络测速、临时文件清理、回收站清空、DNS 刷新、截图录屏、锁屏、任务管理器快捷调用等实用系统功能，并支持阳历与农历生日提醒、节日祝福、日历事件提醒以及随机暖心提示。项目基于 Python 与 PySide6 开发，兼容 Windows、Linux 与 macOS，支持低功耗运行、配置存档、自定义数据目录与多套视觉主题，是一款兼顾颜值、陪伴感与生产力的轻量化桌面助手。
